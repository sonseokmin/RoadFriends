# Roadfarm 서버

이 서버는 Roadfarm 모바일 앱의 백엔드 API를 제공하며, MySQL 8.0 (Community Edition)을 기반으로 합니다.

## 사용 기술

- Node.js / Express
- MySQL 8.0.42 (Ubuntu 패키지)
- 기타 라이브러리: dotenv, cors 등
